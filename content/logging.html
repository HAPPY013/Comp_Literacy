Windows Basics

The Microsoft Windows Operating System that is installed on most computers. An operating system is the software that runs your computer and makes it think.
Application software (a.k.a. programs) such as a word processing, spreadsheet programs, or games runs on top of the operating system.
The first screen you see on the monitor when the computer starts up is called the desktop.

Logging In and Out of A Computer

If you're logged on to Windows 10, 8, 7, or Vista, Ctrl+Alt+Del starts Windows Security, which lets you lock the computer, switch to a different user, log off,
start Task Manager, or shut down/reboot the computer.
In Windows XP and prior, the keyboard shortcut just starts Task Manager.
